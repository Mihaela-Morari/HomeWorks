1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть
сделано предварительно?

В GitBash входем в папку в которой будет сохранятся репозиторий:
cd ~/Tel_Ran/Git (моя папка)
Потом переносим репозитории:
git clone link ssh (линк берём с github.com из кнопки CODE )
Переносим файл в индексное пространство:
git add filename ( пишем имя файла)
git commit - m “ коментарий”

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его
отправить на github.com?

git init
git add .
git commit “ ”
Потом создаём в GitHub.com новый репозиторий ( пустой без фаила README)
git remote add origin link repository

3. Kак получить на локальный компьютер изменения с github.com

Через git pull

4. Kакую информацию нам дают команда :

git status --Показывает скатус и информацию о файле (в коком статусе он находится)
git log --Показыват какие commit были созданы и информацию о них.
git remote -v --Проверяем если папка ивляется репозиторием для github.
